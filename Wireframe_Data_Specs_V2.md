
# ✨ 보배시즌 - SSI & 시스템 신뢰도 와이어프레임 데이터 사양 v2.0

## 목표: 인터랙티브 시각화를 위한 원본 데이터 구조 확정
## 핵심 컨셉: 불안감 유발(문제 정의) → 시스템 감시(데이터 증명) → 통제권 확보(해결책 제시)

### 📊 섹션 A: KPI 대시보드 (시스템 신뢰도)
1. DPSR Score (핵심 지표): [0-100%] / Deep Copper 강조 필수.
2. Uptime Rate (지속성): [N시간/월] / Clear Sky Blue 그래프.
3. Anomaly Count (감시 기능): [일별 꺾은선 그래프] - 마우스 오버 시 상세 오류 로깅 팝업 연결.
4. Latency Avg: [X ms]

### 📈 섹션 B: 개인 리스크 보고서 (SSI)
1. Current Risk Level: [Low/Medium/High] / 색상 경고 시스템 적용.
2. Mitigation Gap: [% 부족] - 목표 대비 격차를 시각화하는 '진행 막대' 구현 필수.
3. Expected Improvement: [% 개선 예상] - 해결책 제시 섹션에서 가장 크게 강조 (Deep Copper).
4. Actionable Steps: [체크리스트 3~5개].

### 🎨 디자인/인터랙션 가이드라인 요약
*   **Primary Color:** Midnight Blue (`#0A1931`) - 배경, 권위.
*   **Secondary Color:** Deep Copper (`#B8860B`) - CTA 및 최종 KPI 강조 (문제 해결 지점).
*   **Accent Color:** Clear Sky Blue (`#ADD8E6`) - 시스템 연결선, 데이터 흐름(Flow) 시각화.
*   **인터랙션 규칙:** 모든 데이터 변동은 '시스템 감시'의 결과로 인식되어야 함. (라인/점진적 노출).
